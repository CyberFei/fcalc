# fcalc

Floating-Point Calculation
用于 JS 浮点数运算

### 引入

```java
import "FC" from "path/to/fcalc"
```

### 使用

```javascript
// 加法
FC.add(0.1, 0.2, 0.3); // 0.1 + 0.2 + 0.3 = 0.6

// 减法
FC.sub(0.1, 0.2, 0.3); // 0.1 - 0.2 - 0.3 = -0.4

// 乘法
FC.mul(0.1, 0.2, 0.3); // 0.1 * 0.2 * 0.3 = 0.006

// 除法
FC.div(0.3, 0.2, 0.1); // 0.1 / 0.2 / 0.3 = 15
```
